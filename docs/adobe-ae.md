Adobe After Effects 键盘快捷键
===

Adobe After Effects 中键盘快捷键的可视化备忘单

键盘快捷键
-----
<!--rehype:body-class=cols-2-->

### 常规

| 快捷键 | 说明 |
| --- | --- |
| `Ctrl` `Alt` `N` | 新建项目 |
| `Ctrl` `O` | 打开项目 |
| `Ctrl` `Alt` `Shift` `P` | 打开最近的项目 |
| `Ctrl` `Alt` `Shift` `N` | 在“项目”面板中新建文件夹 |
| `Ctrl` `Alt` `Shift` `K` | 打开“项目设置”对话框 |
| `Ctrl` `F` | 在“项目”面板中查找 |
| `Alt` + 单击位深度按钮 | 循环切换项目的色位深度 |
<!--rehype:className=shortcuts left-align-->

### 项目
<!--rehype:wrap-class=row-span-2-->

| 快捷键 | 说明 |
| --- | --- |
| `Ctrl` `A` | 全选 |
| `Return` | 重命名选中的图层、合成、文件夹、效果、组或蒙版 |
| `Enter`（数字小键盘） | 打开选中的图层、合成或素材项目 |
| `Ctrl` `Alt` `▼` / `▲` | 按堆积顺序向下（向后）或向上（向前）移动选中的图层、蒙版、效果或者渲染项目 |

| `Shift` `▼` / `▲` | 将选择项扩展到“项目”面板、“渲染队列”面板或者“效果控件”面板中的下/上一个项目 |
| `Ctrl` `D` | 复制选中的图层、蒙版、效果、文本选择器、动画制作工具、操控网格、形状、渲染项目、输出模块或者合成 |
| `Ctrl` `Q` | 退出 |
| `Ctrl` `Z` | 撤消 |
| `Ctrl` `Shift` `Z` | 重做 |
| `Ctrl` `Alt` `/`（数字小键盘） | 清理所有内存 |
| `Esc` | 中断运行脚本 |
| `Ctrl` `Alt` `E` | 在“信息”面板中显示与当前时间的帧所对应的文件名 |
