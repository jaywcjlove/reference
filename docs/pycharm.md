PyCharm 键盘快捷键备忘清单
===

这个 [PyCharm](https://resources.jetbrains.com/storage/products/pycharm/docs/PyCharm_ReferenceCard.pdf) 快速参考备忘单显示了它的键盘快捷键和命令

### 创建和编辑

:-- | --
:-- | --

`Ctrl` + `O` / `Ctrl` + `I` | 覆盖/实施成员
`Ctrl` + `Alt` + `T` | 围绕着……

`Ctrl` + `NumPad` `+` / `-` | 展开/折叠代码块

`Ctrl` + `S` | 保存全部
