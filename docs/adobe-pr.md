### "节目监视器"面板

#### 注意

要对图形图层使用微调，请在单个图形中选择至少一个图层（会出现蓝色方框）。 节目监视器处于焦点中。

| 快捷键 | 说明 |
| --- | --- |

| `Shift` `Ctrl` `→` | 将选定对象向右微移五帧 |
| `Shift` `Ctrl` `←` | 将选定对象向左微移五帧 |
| `Shift` `Ctrl` `↓` | 将选定对象向下微移五帧 |
| `Ctrl` `↑` | 将选定对象向上微移一帧 |
| `Ctrl` `→` | 将选定对象向右微移一帧 |
| `Ctrl` `←` | 将选定对象向左微移一帧 |
| `Ctrl` `↓` | 将选定对象向下微移一帧 |
