let vs var

  // 这是“let”的最大范围
  // i 可以访问 ✔️

// i 不能访问 ❌

  // i 可以访问 ✔️

// i 可以访问 ✔️

var 的范围是最近的函数块，而 let 的范围是最近的封闭块。

for (let j = 0; j < 3; j++) { 
  setTimeout(_ => console.log(j), 10);
}
