VSCode 备忘清单
===

这个 VSCode (Visual Studio Code) 快速参考备忘单显示了它的键盘快捷键和命令。

## Windows

### 一般的

| -                           | -                      |
|-----------------------------|------------------------|
| `Ctrl` `Shift` `P` _/_ `F1` | 显示命令面板             |
| `Ctrl` `P`                  | 快速打开，转到文件        |
| `Ctrl` `Shift` `N`          | 新窗户                  |
| `Ctrl` `Shift` `W`          | 关闭窗口                |
| `Ctrl` `,`                  | 用户设置                |
| `Ctrl` `K` `Ctrl` `S`       | 键盘快捷键               |
